# Typography

## Hierarchy

- Keep the type scale small and intentional.
- Use size and weight to establish hierarchy.
- Avoid more than 3 type sizes on a single screen.
- Avoid more than 2 typefaces in one product.

## Alignment

- Body copy and headings should be left-aligned by default.
- Numbers in tables should be right-aligned.
- Button labels should be centered.

## Line length and line height

- Keep body copy around 60-80 characters per line.
- Constrain long desktop paragraphs to a readable measure.
- Use tighter line height for headings and looser line height for body text.

## Font weight

- Use regular, medium, and semibold for most products.
- Reserve bold or heavier weights for display moments.
- Do not use italics for normal UI labels.

## Indentation in displayed content

- Use spaces, not tabs, in any user-facing code or structured text.
- Keep indentation consistent within the product.
