# Spacing

## Scale

- Use a fixed spacing scale throughout the product.
- Prefer common increments like 4px or 8px.
- Never use arbitrary values like 13px or 17px.

## Proximity

- Put related elements closer together than unrelated ones.
- Increase spacing as you move from inner elements to outer containers.
- Keep padding and margins intentional and consistent.

## Touch and cursor spacing

- Keep touch targets large enough for fingers.
- Maintain enough space between adjacent touch targets to avoid mis-taps.
- Cursor-based interactions can be smaller, but still need clear affordance.
