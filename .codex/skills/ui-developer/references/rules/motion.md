# Motion

## Purpose

- Animate for orientation, feedback, continuity, focus, or delight.
- Do not animate for decoration alone.

## Duration

- Micro-interactions: 100-150ms.
- UI transitions: 200-300ms.
- Page transitions: 300-400ms.
- Complex sequences: 400-600ms.
- Never exceed 600ms.

## Easing

- Use ease out for entering elements.
- Use ease in for exiting elements.
- Use ease in-out for movement within the screen.
- Use spring sparingly for playful interactions.

## Reduced motion

- Respect prefers-reduced-motion.
- Provide non-motion alternatives.
- Avoid flashing or seizure-triggering effects.
