# Data Display

## Numbers

- Right-align numbers in tables.
- Use thousands separators.
- Keep decimal precision consistent within a column.
- Show zero explicitly.

## Dates and times

- Use relative time for recent events.
- Use absolute dates for older entries.
- Include time where precision matters.
- Respect the user's local time zone.

## Empty and null values

- Use an em dash for no value.
- Use 0 for zero counts.
- Use N/A for not applicable.
- Never leave a field blank without meaning.

## Truncation

- Use ellipsis when text is cut off.
- Show full value on hover where appropriate.
- Truncate paths and identifiers in the middle when needed.
- Do not truncate numbers.
