# Icons

## Rules

- Pair icons with labels unless the icon is universally understood.
- Use one icon style throughout the product.
- Keep icon sizes consistent.
- Do not hardcode icon colors outside the design tokens.

## Common sizes

- Inline with body text: 16px.
- Labels and buttons: 16-20px.
- Navigation: 24px.
- Feature or empty-state icons: 32-48px.
- Hero illustrations: 64px+.
