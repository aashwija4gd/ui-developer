# Quick Reference

- Numbers in tables: right-aligned.
- Text in tables: left-aligned.
- Links: underlined plus distinct color.
- Touch targets: minimum 44x44px on mobile.
- Hover states: required on desktop, not enough on mobile.
- Empty cells: use an em dash.
- Primary buttons: one per screen or section.
- Focus states: never remove.
- Contrast: keep body text readable.
- Destructive actions: confirm before executing.
- Empty states: include icon, heading, and CTA.
- Loading states: never show a blank screen.
