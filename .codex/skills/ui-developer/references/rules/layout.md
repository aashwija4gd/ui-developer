# Layout

## Grids

- Use a grid for every layout.
- Mobile: 4 columns, 16px margin, 16px gutter.
- Tablet: 8 columns, 24px margin, 24px gutter.
- Desktop: 12 columns, 80-120px margin, 24px gutter.
- Align all content to the grid.

## Balance

- Make primary actions visually prominent.
- Use whitespace to separate unrelated content.
- Avoid screens where every element has equal weight.

## Depth

- Use layering intentionally: background, surface, overlay, toast.
- Communicate depth with surface color, shadow, blur, and borders.

## Responsive behaviour

- Design mobile first.
- Let lower-priority content collapse or move below the fold on small screens.
- Switch desktop sidebar navigation to mobile-friendly patterns.
- Handle wide tables with horizontal scroll or alternative layouts.
